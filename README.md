# Predicting_Win_Probability_Momentum_Shifts_T20

# 🏏 Cricket Win Probability & Momentum Shift Analysis

## 📌 Objective
To predict win probability during a live cricket match and identify key momentum shifts using machine learning and data visualization.

---

## ⚙️ Approach
- Collected ball-by-ball match data from Cricsheet  
- Engineered features such as runs, wickets, overs, and required run rate  
- Trained a machine learning model to estimate win probability dynamically  
- Simulated match progression and plotted probability curves  
- Detected momentum shifts using sharp probability changes  

---

## 📊 Results
- Successfully modeled real-time win probability trends  
- Identified key turning points (momentum shifts) in matches  
- Built visualizations showing probability evolution over time  

---

## 📂 Dataset
The project utilized open-source, ball-by-ball T20 International data sourced from [Cricsheet](https://cricsheet.org/downloads/#experimental).

Note that as this dataset updates continuously, the model was trained on a specific snapshot of **336,000+ deliveries accessed in April 2026**.  
The latest dataset can be downloaded directly from the above link.
